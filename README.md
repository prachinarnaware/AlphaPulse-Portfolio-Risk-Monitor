 AlphaPulse-Portfolio-Risk-Monitor
Portfolio Risk Analytics using Monte Carlo Simulation (Python &amp; Tableau).# 📈 AlphaPulse: Portfolio Risk & Volatility Monitor

 Project Overview
This project was developed during my **Data Analytics Internship at Infotact Solutions**. It focuses on assessing the risk of a multi-asset portfolio (Gold, JPM, Tesla) using **Monte Carlo Simulations**.

 Tech Stack
- **Language:** Python (Pandas, NumPy, Matplotlib)
- **Visualization:** Tableau Desktop
- **Methodology:** Monte Carlo Simulation, Value at Risk (VaR), Correlation Analysis

 Key Features
- **Simulation:** 10,000 trials to predict portfolio growth.
- **Risk Metrics:** Calculated 95% Value at Risk (VaR).
- **Dashboard:** Interactive Tableau dashboard showing:
  - Asset Performance Trends
  - Risk Clusters (Volatility)
  - Correlation Heatmap
  - Monte Carlo Distribution (Histogram)

 Files Included
- `portfolio_risk_analysis.ipynb`: Python code for simulation.
- `AlphaPulse_Dashboard.twbx`: Tableau workbook.
- `Dashboard_Screenshot.png`: Visual overview of the project.
 Conclusion
The analysis demonstrates that the portfolio remains diversified with a manageable daily risk level of $184.13 at 95% confidence.
